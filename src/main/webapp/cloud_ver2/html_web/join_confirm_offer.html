공급(판매) 권한 신청

공급자 권한 승인 절차

  1. 1. 가입 승인을 위한 구비서류를 정보입력 스텝에서 파일로 업로드
  2. 2. 구비서류 업로드 후 서명 날인을 대체하여 기업공인인증서 인증
  3. 3. 한국정보화진흥원에서 첨부 서류 확인 및 승인
  4. 4. 승인 완료 메일 수신 이후 공급자 권한으로 서비스 정상 이용 가능(첨부파일 업로드후 영업일 기준 최대 48시간 이내 승인)

※ 구비서류를 PDF파일로 준비하여 정보입력시 업로드 해 주세요.
