• 01. 전문가상담 서비스란?

    구매자(수요자)가 찾고자 하는 클라우드 서비스가 없을 경우 또는 어떤 서비스를 선택해야 하는지 모를 때,
    구매자(수요자)가 찾고자 하는 서비스를 등록하면 판매자(공급자)들과
    매칭시켜주는 전문가상담 서비스를 제공합니다.

  • 02. 전문가상담 등록/선정

    찾고자 하는 클라우드 서비스 내용을 등록 후 판매자나 회원분들이 댓글을 올려주시면
    수요자 회원 분들은 본인이 찾고자하는 내용과 일치하는 댓글을 등록하신 판매자의 댓글을 선정해주시면
    판매자 회원이 직접 연락을 드릴 수 있습니다.
